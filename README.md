The Spam Email Classifier is a machine learning project designed to automatically detect and filter out unwanted or malicious emails (spam) from legitimate messages (ham). The goal is to build an intelligent system that can analyze email text and accurately determine whether it is spam or not, helping users maintain a clean and secure inbox.

To develop a supervised machine learning model that classifies emails (or SMS messages) as Spam or Not Spam (Ham) based on the textual content. The system should achieve high accuracy, recall, and precision to minimize false positives (ham marked as spam) and false negatives (spam marked as ham).

Dataset taken from kaggle.
